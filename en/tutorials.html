What will you learn today?

How do the web and the Internet work?
#web

Did you know that the world-wide web and the Internet are not the same thing? You will learn about this and much more in this course
HTML
#web #frontend

HTML is a language that gives structure to your websites. Without it, browsers would not be able to lay out the content on a webpage or distinguish a text from an image!
CSS
#web #frontend

CSS is a language that provides styling and decoration for your HTML. With it, you can convert a boring clickjack into a fancy button everyone wants to click on. You will also be able to better organize the layout of a website with more attractive designs
JavaScript
#web #frontend

JavaScript is the programming language of the web. What you can do with a normal programming language, you can do it with JavaScript. Not only that, but with JavaScript you can also manipulate HTML and CSS. Like that, your websites stop being static and begin being dynamic
Think like a programmer
#programming

Programming is not simply learning the syntax of a programming language by root or throw chunks of messy, colorful code on the screen. In this course, you will discover how programming is its own way of thinking
Python
#programming

Python is a programming language not only useful for those who want to learn to program without getting overwhelmed by thousands of concepts. It is also a perfect tool for getting into the fields of data analysis, data science, IoT or even the web
Android (with Java)
#android #java

Android is an operating system that is not only used in smartphones but also in smartwatches, tablets, TVs and even cars. In this Android course you will learn how Android works and get to work on your first simple Android apps using Java
