Was willst du heute lernen?

Wie funktionieren das Web und das Internet?
#web

Wusstest du, dass das WWW und das Internet nicht das Gleiche sind? Dieses und vieles mehr wirst du in diesem Kurs lernen können!
HTML
#web #frontend

HTML ist die Sprache, die deinen Webseiten eine Struktur gibt. Ohne HTML können Browser die Ausrichtung und Anordnung der Inhalte nicht interpretieren. Sie können selbst nicht zwischen einem Text und einem Bild unterscheiden!
CSS
#web #frontend

CSS ist die Sprache, die deinem HTML schöner und benutzerfreundlicher gestaltet. Damit kannst du einen langweilig aussehenden Knopf in einen attraktiven Button verwandeln, die jeder anklicken möchte. Mit CSS kannst du auch die Anordnung der Webseite besser steuern und schöner entwerfen
JavaScript
#web #frontend

JavaScript is the programming language of the web. What you can do with a normal programming language, you can do it with JavaScript. Not only that, but with JavaScript you can also manipulate HTML and CSS. Like that, your websites stop being static and begin being dynamic
Denke wie ein Programmierer
#programmieren

Programmieren heißt nicht einfach die Syntax einer Programmiersprache auswendig zu lernen, oder unüberlegten Code auf den Bildschirm auszuspucken. In diesem Kurs wirst du erfahren, wie das Programmieren eine Art Denken für sich ist
Python
#programmieren

Python ist eine Programmiersprache, die nicht nur für diejenigen praktisch ist, die ein bisschen in die Programmierwelt eintauchen wollen ohne viele Einzelheiten vorab lernen zu müssen. Es ist zudem ein perfektes Werkzeug, um in die Bereiche der Datenanalyse, Data Science, IoT oder sogar dem Web reinzuschnuppern
Android (with Java)
#android #java

Android ist ein Betriebssystem, das nicht nur in Smartphones, sondern auch in Smartwatches, Tablets, Fernsehern und sogar in Autos Verwendung findet. In diesem Android-Kurs wirst du lernen wie Android funktioniert und mit deinen ersten einfachen Apps Erfahrung sammeln und in Java entwickeln
